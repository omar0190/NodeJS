Functions

Functions are first class citizens in Node's JavaScript, similar to the browser's JavaScript. A function can have attributes and properties also. It can be treated like a class in JavaScript.

function myFirstFunction(){ return "My first Greeting"}
Callback functions

Callback function is a frequent use in Nodejs. An example of this would be for you as a user to view this page:

app.get("/snippets", (req, res) => { res.sendFile(__dirname + "/directory/snippets/snippets.html"); });

A callback function is a function which is accessible by another function and is invoked after the first function, if that function completes.

Strict

The purpose of "use strict" is to indicate that the code should be executed in "strict mode". With strict mode, you can not, for example, use undeclared variables.

Scopes

Scope in JavaScript refers to the current context of code, which determines the accessibility of variables to JavaScript. The two types of scope are local and global: Global variables are those declared outside of a block. Local variables are those declared inside of a block
