Terminal Commands


Github

The first things we learned in this class was the common github commands

To add:

git -add, adds a change in the repository you are working on.

git -commit -m "insert comment here", commits the code to github

git push, pushes the code you've added to the repository

To pull:

git -pull, pulls the code from the repository.

Express and nodemon

The framework we are working with in NodeJS is primarily Express.js.
Express is also used to serve our static files in Node.
We are also using nodemon, which is a tool to always run the code without typing it manually
Npm stands for Node Package Manager and is the default package manager for Node.js.

To install:

npm install express
npm install nodemon

package.json

A package. json is a JSON file that exists at the root of a Javascript/Node project.
It holds metadata relevant to the project and it is used for managing the project's dependencies, scripts, version and a whole lot more.

To run program

To run the program manually we could use the command node {programname.js} or nodemon {programName.js}

we could also make our own command to run the program by creating a script in package.json
